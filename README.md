# Git-Study

最简单的从0开始提交代码到服务器

```
echo "# Git-Study" >> README.md //将内容写入MD文件
git init                        //初始化git仓库
git add README.md               //将md文件添加到暂存区
git commit -m "first commit"    //将文件提交到本地仓库
git branch -M master            //重命名分组
git remote add origin https://github.com/1535892832/Git-Study.git //关联远程分支
git push -u origin master //推送到远程仓库
```
